# 🔐 Password Strength Checker

A Python command-line application that analyzes password strength and provides feedback to help users create more secure passwords.

---

## 📖 Overview

This tool checks whether a password is strong based on common security practices such as:

- Minimum length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

It also provides suggestions for improving weak passwords.

---

## ✨ Features

- 🔍 Password strength analysis
- 🔒 Detects weak passwords
- 💡 Security recommendations
- 🎨 Colorful terminal output
- 🧰 Beginner-friendly Python project

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/adebayofunsho22-dot/password_strength_checker.git
```

Go into the project:

```bash
cd password_strength_checker
```

Run the program:

```bash
python password_checker.py
```
